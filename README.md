# Tally.so Clone

A full-stack Tally.so-style form builder with a Next.js frontend and FastAPI backend.

## Project Structure

- frontend: Next.js app with the form builder, dashboard, and public pages
- backend: FastAPI app with authentication, forms, and submissions

## Backend

- Framework: FastAPI
- Local API base: http://127.0.0.1:8000 (or your configured port)
- Deployed API docs: https://tally-so-test-production.up.railway.app/docs

## Frontend

- Framework: Next.js (App Router)
- Deployed site: https://tally-so-test.vercel.app/

## Docs

- Swagger/OpenAPI: https://tally-so-test-production.up.railway.app/docs
